# bf_jit
Brainfuck JIT Runtime

## Usage

```bash
$ ./bfi -h
usage: ./bfi [-h] [-j] [-p] <input-file>
    -h  this help message
    -j  enable JIT compilation
    -p  print compiled bytecode IR
```

## Build

Clone this repository:

```bash
git clone https://github.com/chenzhuoyu/bf_jit
```

Build the interpreter:

```bash
make
```
